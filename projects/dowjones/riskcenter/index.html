Dow Jones Riskcenter

Web App

Web: HTML, CSS, React
Visualizations: D3.js

Design for Dow Jones Riskcenter: a suite of Risk & Compliance web apps consisting of Dashboard, RiskReports, Search, RiskAverter, Trade Compliance.

Also designed and developed front-end of visualization tool for Risk & Compliance database. Allows user to input search query into database which outputs network node of connections to a person or entity, displaying Risk flags, as well as providing a map of their connections.
