Factiva Mobile App Redesign

React Native (Android / iOS)

Design: Sketch, Adobe Illustrator
Prototype: Invision

Factiva is a global news database of nearly 33,000 premium sources, including licensed publications, influential websites, blogs, images and videos. My role was to design the native app version of Factiva for both iOS and Android. This also marks the first-ever React Native app released by Dow Jones.
